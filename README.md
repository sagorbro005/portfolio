## Customization Guide

### 1. Personal Information

Edit `index.html` to update:
- Your name (line 42)
- Hero description (line 49)
- About section content (lines 89-102)
- Statistics (lines 104-118)
- Contact details (lines 299-319)

### 2. Skills

Update the skills section (lines 127-226) with your own:
- Skill categories
- Icons (using Font Awesome)
- Descriptions
- Technology tags

### 3. Projects

Customize your projects (lines 235-382):
- Project titles
- Descriptions
- Technology tags
- Links to live demos and GitHub repos

### 4. Colors

Modify the color scheme in `styles.css` (lines 9-15):
```css
--primary-color: #13ff9d;
--secondary-color: #8b5cf6;
--accent-color: #ec4899;
```

### 5. Typing Animation

Edit the typing text in `script.js` (line 25):
```javascript
const textArray = ['Full Stack Developer', 'UI/UX Designer', ...];
```

### 6. Images

Replace the placeholder icons with your own images:
- Hero image: `.hero-image .image-placeholder`
- About image: `.about-image .image-frame`
- Project images: `.project-placeholder`
